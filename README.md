## **Read Me**

### **1: Spielverlauf**

Das Spiel besteht aus zwei Teilen. Der 1. Teil besteht aus einem Quizz. In dem 2. Teil tritt der Spieler in einer Runde 
Tic-Tac-Toe gegen den Computer an. Um zu gewinnen, müssen **beide** Teile erfolgreich abgeschlossen werden. Viel Erfolg! 

### **2: Spiel starten**

#### 2:1 Editor 

Das vollständige Spiel wird im Python Editor über das **main**-file gestartet. 
Falls nur einzelne Teile gespielt werden möchten, können sowohl **Text-Adventure** als auch **Tic-Tac-Toe** einzeln 
gestartet werden. Es ist darauf zu achten, dass die Datei **use-import** in jenem Fall mitgeladen wird, da hieraus 
Funktionen und Begriffe importiert werden. 

#### 2:2 Terminal/ Shell

Anderenfalls kann das Spiel ebenfalls über das Terminal/ Shell geöffnet weren (Python muss installiert sein).
Hierfür ist am besten ein Ordner anzulegen, der alle Dateien enthält. Daraufhin sind folgende commands einzugeben: 


cd "Pfad/zum/Ordner"

**1. Spiel**  

**macOS/Linux:** python3 main.py |
**Windows:** python main.py

**2. Quizz**  

**macOS/Linux:** python3 quizz.py |
**Windows:** python3 quizz.py

**3. Tic-Tac-Toe**

**macOS/Linux:** python3 tic-tac-toe.py |
**Windows:** python3 tic-tac-toe.py
